Project Overview



This repository contains my Digital Literacy Portfolio created as part of the CSE0001 Digital Literacy course at VIT Bhopal University. The purpose of this project is to develop essential digital skills such as creating professional online profiles, using digital tools, practicing coding platforms, writing professional emails, and understanding cybercrime awareness.



The project is divided into five tasks, each corresponding to a module of the Digital Literacy course. All the tasks, screenshots, documents, and reports are organized in this repository.

Task Summary

Task 1 – Digital Literacy Awareness Infographic



In this task, I created a digital literacy infographic using Canva. The infographic explains what digital literacy is, useful digital tools for students, and safe internet practices. The infographic is included in the task-1-presentation folder.



Task 2 – Student Digital Portfolio



In this task, I created and updated my professional profiles on platforms such as GitHub, LinkedIn, and coding platforms. Screenshots of the profiles are included in the task-2-portfolio folder. These platforms will help me build my professional presence and portfolio during my B.Tech journey.



Task 3 – Coding \& Collaboration Platforms



In this task, I explored coding practice platforms and completed a beginner coding challenge. I also created a Google Form quiz on Digital Literacy Awareness and collected responses. Screenshots and proof of work are included in the task-3-platforms folder.

Task 4 – Professional Email \& Etiquette



In this task, I wrote two professional emails:



Email to a professor requesting assignment extension

Email to an internship coordinator expressing interest in internship



I also created a Social Media Do’s and Don’ts checklist. All files are included in the task-4-email-etiquette folder.



Task 5 – Cybercrime Awareness



In this task, I wrote a case study on cybercrime and created a Stay Safe Online prevention checklist, including UPI safety tips and cybercrime reporting information. All files are included in the task-5-cybercrime folder.



Conclusion



This Digital Literacy project helped me understand the importance of professional online presence, digital communication, online safety, and digital tools for learning and collaboration. This project also helped me create my digital portfolio which will be useful for internships, placements, and academic work in the future.
